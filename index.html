Output Gap Q3: -6.4%

The nowcast of the output gap is based on the method described in Berger, Morley, and Wong (2020). Please cite the paper when referring to the nowcast estimate.

The nowcast for the current quarter in which real GDP has not yet been released is reported in the figure using an orange line from the actual estimated output gap for previous quarters. The actual estimated output gap is based on parameter estimates using data from 1967-2019 for the mixed-frequency Bayesian VAR specified in Berger, Morley, and Wong (2020) and data up to the latest quarter for which real GDP has been released from FRED . The monthly variables that feed into the nowcast are given in the table.

Please see Berger, Morley, and Wong (2020) (full reference below) for more details and please cite the paper when using the method or the results presented here.
